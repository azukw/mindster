## Overview

**Mindster** is a minimalist and elegant web-based logic game where deduction, memory, and strategy are key.  
Each day, a new puzzle challenges players to uncover a hidden color sequence using visual hints and logical reasoning.

Inspired by the classic **Mastermind**, Mindster modernizes the concept with daily challenges, multiple difficulty modes, accessibility options, and detailed progress tracking.

---

## Gameplay

- A secret color combination is generated every day
- You have **up to 8 attempts** to find the correct sequence
- After each attempt, visual clues help you refine your next guess
- The daily puzzle is identical for all players worldwide

## Accessibility & Customization

Mindster is designed to be accessible to as many players as possible:

- Dark mode for comfortable night play
- High contrast mode for improved readability
- Multiple color themes unlocked through progression
- Fully playable with mouse or keyboard

## Languages

Language selection is automatic or manually adjustable.
